# 坐标系 V2：球场中心为原点

## 变更说明
从"近端底线左角为原点"改为"球场中心（网中心地面投影点）为原点"。

## 新坐标系定义
```
原点: 球场中心 (网的正中间地面投影点)
X轴: 从左到右 (站在cam66视角)
Y轴: 从近端到远端 (cam68端 → cam66端)
Z轴: 向上

           cam66 (y=17.164)
              ↓
  (-2.745, 11.885) ─── (2.745, 11.885)   ← 远端底线
  │                     │
  │                     │
  (-2.745, 6.400) ──── (2.745, 6.400)    ← 远端发球线
  │          │          │
  │      (0, 6.400)     │                 ← 中线
  │          │          │
  ═══════════════════════  y = 0          ← 网 (NET)
  │          │          │
  │      (0, -6.400)    │
  │          │          │
  (-2.745, -6.400) ─── (2.745, -6.400)   ← 近端发球线
  │                     │
  │                     │
  (-2.745, -11.885) ── (2.745, -11.885)   ← 近端底线
              ↑
           cam68 (y=-17.163)
```
